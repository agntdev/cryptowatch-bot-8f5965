# CryptoWatch Bot

Telegram bot that privately monitors crypto prices and notifies users on configured price thresholds or percent-change alerts, with quiet hours, cooldowns, and optional morning summaries.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
